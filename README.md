# Assessoria Veritas - Landing Page

Landing page desenvolvida em React + Vite para o cliente **Assessoria Veritas**, especializado em edição de vídeo profissional. Projeto gerado pela base44 e refinado manualmente, com design premium em tema escuro, Tailwind CSS e animações em Framer Motion.

## Tecnologias

- **React** 18.3.1
- **Vite** 6.0.5
- **Tailwind CSS** 3.4.18
- **Framer Motion** 11.15.0
- **Lucide React** 0.469.0

## Estrutura do Projeto

```
/Componentes
  /Pousar - Componentes da landing page
    - secaoDeHerois.jsx
    - secaoDeBeneficios.jsx
    - secaoDeResultados.jsx
    - secaoDeServicos.jsx
    - secaoDeDepoimentos.jsx
    - secaoDePrecos.jsx
    - secaoDeCTAs.jsx
  /ui - Componentes de UI reutilizáveis
    - button.jsx
/pages
  - Lar.jsx (página principal)
```

## Design

- Tema escuro premium
- Paleta de cores dourada (#C9A24D, #E6C97A)
- Animações suaves com Framer Motion
- Design responsivo (mobile-first)
- Foco em conversão
